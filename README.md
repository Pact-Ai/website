# Pact-Ai Website

The official website for **Pact-Ai** — building AI systems that help, matter, and scale. Specializing in Natural Language Processing, Computer Vision, and breakthrough solutions for low-resource languages.

## Tech Stack

| Layer       | Technology                          |
| ----------- | ----------------------------------- |
| Framework   | [Next.js 15](https://nextjs.org/) (Pages Router) |
| UI Library  | [HeroUI](https://heroui.com/)       |
| Styling     | [Tailwind CSS v4](https://tailwindcss.com/) + [tailwind-variants](https://www.tailwind-variants.org/) |
| Animations  | [Framer Motion](https://motion.dev/) |
| Language    | TypeScript                          |
| Fonts       | Inter (sans), Fira Code (mono) via `next/font/google` |

## Getting Started

### Prerequisites

- **Node.js** ≥ 18
- **npm** ≥ 9

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Opens the site at [http://localhost:3000](http://localhost:3000) with Turbopack for fast refresh.
